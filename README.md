## 🇺🇸 English  

<br />

### Description  
Enables IPv6 on Linux-based VPS or Dedicated Servers.  

<br />

### Compatibility  
| Distribution    | Minimum Version |
|-----------------|-----------------|
| Ubuntu          | 18.04           |
| Debian          | 10              |
| Fedora          | 32              |
| CentOS          | 7               |
| AlmaLinux       | 8.3             |
| Rocky Linux     | 8.3             |


### Prerequisites  
• Root access  
• `wget` installed  
• `bash` version 4.3 or higher  


### Usage  
```bash
wget -qO- https://raw.githubusercontent.com/neoSmartness/enableIPv6/main/enableIPv6.sh | bash -s -- --lang en
```
